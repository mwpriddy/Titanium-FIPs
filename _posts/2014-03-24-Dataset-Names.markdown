The datasets are named to correspond with the peaks and valleys on the load cycle during fatigue.


|Figure    |File   |
|----------|-------|
| Peak 1   | c1max |
| Peak 2   | c2max |
| Peak 3   | c3max |
| Valley 1 | c1min |
| Valley 2 | c2min |
| Valley 3 | c3min |
